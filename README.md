## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/felipegfsouza/network-admin-tools.git
```

Entre no diretório do projeto

```bash
  cd network-admin-tools 
```

De a permissão 

```bash
 chmod +x script.sh
```

Para rodar

```bash
 ./filename.sh
```
